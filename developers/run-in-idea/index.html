使用IDEA运行FullNode节点¶

该文档旨在方便有一定经验的开发者在集成开发环境IDEA中运行FullNode节点 以下内容针对java-tron的master分支

一、配置集成开发环境IDEA¶

IDEA配置¶

  • Oracle JDK 1.8 当前不支持OpenJDK

  • 安装Lombok插件

  • 将Compiler下Annotation Processors中的Enable annotation processing前打勾

二、部署指南¶

1.创建目录¶

/deploy

mkdir deploy

2.克隆最新的代码¶

git clone https://github.com/tronprotocol/java-tron

3.切换master分支¶

cd java-tron
git checkout -t origin/master

4.编译代码¶

./gradlew build

编译过程可能需要一定时间，请耐心等待。 若编译成功，你可以看到类似如下信息：

如果不想执行单元测试任务，可运行如下命令：

./gradlew build -x test

5.启动FullNode节点¶

编译成功后，可通过java-tron/src/main/java/org/tron/program/FullNode.java路径找到主函数文件，启动一个全节点。
